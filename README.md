# study-env-project
A RL OpenEnv environment simulating a student study planner with step(), reset(), and reward based learning. 
# Study Environment (OpenEnv Project)

## Description
This project is a simple reinforcement learning environment that simulates a student study planner.

## Actions
- study → increases progress
- waste → no progress

## State
- Represents study progress

## Reward
- study → +1
- waste → 0

## Levels
- Easy → target 3
- Medium → target 5
- Hard → target 8

## How to Run
Run agent.py to see the environment working.
